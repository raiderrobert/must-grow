# I must grow.

A browser-based space game where you start as a tiny station orbiting Earth and grow into a solar-system-devouring force of nature. Destroy debris, consume mass, evolve through tiers, and devour every planet — including the Sun.

## How to Play

- Orbit Earth and destroy nearby debris to collect mass
- Earn upgrades as you consume more mass — each tier makes you exponentially stronger
- Escape orbit and travel to other planets to feed
- Destroy all 8 planets and the Sun to win

## Controls

| Action | Keyboard | Gamepad |
|--------|----------|---------|
| Move | WASD / Arrows | Left Stick |
| Burst Fire | Space | A |
| Boost | Shift | RT |
| Settings | ESC | Start |

## Development

```bash
pnpm install
pnpm dev        # start dev server
pnpm test       # run tests
pnpm build      # type check + production build
```

## Tech Stack

- **Phaser 3** — game engine
- **TypeScript** — language
- **Vite** — bundler
- **Vitest** — tests
- **Cloudflare Pages** — hosting
